1. 🕵️‍♂️ Random User Generator
What it does: Fetch a random user profile and display their name, email, picture, etc.

🔗 API: https://randomuser.me

📚 Skills: fetch, DOM manipulation, JSON parsing

fetch('https://randomuser.me/api/')
  .then(res => res.json())
  .then(data => {
    const user = data.results[0];
    console.log(user.name.first);
  });
